Togglish

This thing show how you can use radio or checkbox :checked to toggle properties on a sibling element without using any JavaScript.

The HTML and CSS footprints are minimal:

Markup

<div class="togglish">
  <input id="togglish-check" type="checkbox" />
  <label for="togglish-check">Toggle</label>
  <div class="target">
    ...
  </div>
</div>

CSS

.togglish {
    --togglish-display: none; 
  }
.togglish .target {
  display: var(--togglish-display); 
}
.togglish > input[type='checkbox']:checked ~ label + .target,
.togglish > input[type='radio']:checked ~ label + .target {
  --togglish-display: block; 
}

Thats it - you are good to go. Check out the examples below

Although the footprint is small, I have put together a SASS mix-in:

@import "mixins/togglish-helper.scss";

$a: ( 
  vars: ( --togglish-display: none ),
  vars-checked: ( --togglish-display: block ),
  target: ( display: var(--togglish-display) )
);    

.togglish {
  @include togglish-helper(
    map-get($a, vars), 
    map-get($a, vars-checked), 
    map-get($a, target)
  );
}

Pros

  • Minimal setup
  • No javascript
  • Easily change to fit any toggle need

Cons

  • Strict markup sequence
  • Requires a unique id on toggle element(checkbox or radio)
  • Requires a for reference to id on toggle element label

Markup

Minimal setup requeres 1 parent element and 3 child elements:

<div class="togglish">
  <input id="togglish-check" type="checkbox" />
  <label for="togglish-check">Toggle</label>
  <div class="target">
    ...
  </div>
</div>
